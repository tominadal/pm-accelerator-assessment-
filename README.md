# PM Accelerator - Tech Assessment (Dual Role)

**Candidate:** Tomás Ignacio Nadal  
**Role Applied:** AI Engineer Intern & Data Science Intern (Dual Role)

This repository contains the technical assessment submissions for the PM Accelerator Internship. The project is split into two distinct areas demonstrating proficiency in both Frontend Engineering and Data Science/Machine Learning.

## Project Structure

### 1. [Frontend Engineering (Weather App)](./frontend/weather-app/README.md)
A responsive, production-ready React web application that interacts with real-world APIs to deliver current weather conditions and 5-day forecasts. 
- **Tech Stack:** React, Vite, Vanilla CSS, Lucide React.
- **Key Features:** Native Geolocation, Open-Meteo API integration (Geocoding & Forecast), strict error handling, and responsive design.
- **[View Frontend Details & Instructions](./frontend/weather-app/README.md)**

### 2. [Data Science & Machine Learning (Weather Trend Forecasting)](./data_science_backend/README.md)
An advanced data analysis and forecasting project built on the Global Weather Repository dataset. 
- **Tech Stack:** Python, Pandas, Scikit-Learn, Prophet, XGBoost, Jupyter Notebook.
- **Key Features:** Statistical anomaly detection, Exploratory Data Analysis (EDA), spatial heatmaps, time-series forecasting, and multivariate temperature prediction.
- **[View Data Science Details & Instructions](./data_science_backend/README.md)**
